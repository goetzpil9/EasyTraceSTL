# EasyTraceSTL -> https://goetzpil9.github.io/EasyTraceSTL/

A free, offline & super simple "CAD" / design tool for 3D printing makers. 

## How it works:
1. 🖼️ **Import** any picture (ideal one with a ruler), set the scale, crop the picture
2. ✍️ **Trace** the countur lines or use geometric shapes
3. ➕ **Modify** shapes (add/subtract like TinkerCAD), smooth corners, ...
4. 🖨️ **Export** your print-ready STL file

*No installation required, but the installed PWA runs 100% offline in your browser.*

Installation (not necessary):
1. Open the page (https://goetzpil9.github.io/EasyTraceSTL/) in your browser.
2. An **Install** icon will appear in the address bar (or, in the menu,
   ‘Install app’). Click on it.
3. EasyTraceSTL will then run in its own window with its own icon – just like a
   desktop app, without the browser toolbar.
4. The first time you launch it online, the browser downloads everything once.
   After that, the tool works even without an internet connection.
