# EasyTraceSTL

A free, offline & super simple "CAD" tool for 3D printing makers. 

## How it works:
1. 🖼️ **Import** any picture (ideal one with a ruler), set the scale, crop the picture
2. ✍️ **Trace** the countur lines or use geometric shapes
3. ➕ **Modify** shapes (add/subtract like TinkerCAD)
4. 🖨️ **Export** your print-ready STL file

*No installation required — runs 100% offline in your browser.*
